# 03 - MLP MNIST Training Loop

## Goal

By the end of this session, participants should be able to run and explain a basic PyTorch training loop.

This lesson uses the Kaggle MNIST MLP notebook only as inspiration. The workshop implementation should run locally with PyTorch and `torchvision.datasets.MNIST`.

## Preparation

Review:

- tensors and shapes from session 02
- the neural network intuition videos in [02 - Tensors and PyTorch](../02-tensors-and-pytorch/README.md)

## Live Session Outline

1. Load MNIST locally with `torchvision`.
2. Build a small multilayer perceptron.
3. Define a loss function and optimizer.
4. Run the training loop.
5. Evaluate accuracy on a validation or test split.
6. Modify one part of the model or training settings.

## Exercises

Exercises will focus on completing parts of the training step: forward pass, loss calculation, backpropagation, optimizer step, and accuracy calculation.
