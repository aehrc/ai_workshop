## Goal

By the end of this session, participants should be able to use Hugging Face tokenizers, datasets, pretrained models, and the model hub.

## Live Session Outline

1. Load a tokenizer.
2. Tokenize text and inspect token IDs.
3. Load a small pretrained model.
4. Run inference locally.
5. Load a small dataset.
6. Understand model cards and hardware expectations.

## Exercises

Exercises will focus on tokenization, decoding, and running inference with a small pretrained model.
