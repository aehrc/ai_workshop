# 06 - Generative Models

## Goal

By the end of this session, participants should be able to run a small pretrained generative model and adjust generation settings.

## Live Session Outline

1. Load a small generative model.
2. Generate text from a prompt.
3. Compare greedy decoding and sampling.
4. Adjust generation parameters such as max tokens and temperature.
5. Discuss quality, speed, and hardware tradeoffs.

## Exercises

Exercises will focus on changing prompts and generation settings, then comparing outputs.
