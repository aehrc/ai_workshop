# 04 - Training Loop Improvements

## Goal

By the end of this session, participants should be able to improve a basic training loop without changing the model idea.

## Live Session Outline

1. Separate training and evaluation functions.
2. Track loss and accuracy over time.
3. Add validation.
4. Save and load checkpoints.
5. Plot simple training curves.

## Exercises

Exercises will focus on adding validation, recording metrics, and saving model checkpoints.
