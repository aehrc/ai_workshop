# 07 - Fine-Tuning Generative Models

## Goal

By the end of this session, participants should understand the shape of a fine-tuning workflow for a small generative model.

## Live Session Outline

1. Prepare a tiny text dataset.
2. Tokenize examples for training.
3. Reuse the training loop ideas from earlier sessions.
4. Run a short CPU-friendly fine-tuning demonstration.
5. Compare outputs before and after fine-tuning.

## Exercises

Exercises will focus on dataset formatting, tokenization, and identifying the parts of the training loop that differ from MNIST.
